# Mazalcare - Premium Shopify Theme

![Mazalcare Store Preview](./mazalcare-com.png)

Mazalcare is a premium Shopify theme designed for skincare and wellness brands. It features a suite of high-performance custom sections tailored for high conversion rates, modern aesthetics, and seamless user experiences across all devices.

## 🚀 Key Features

- **Performance Optimized**: Built for speed with lazy loading and optimized assets.
- **Fully Responsive**: Mobile-first design ensuring a premium look on iPhones and Android devices.
- **Highly Customizable**: Granular control over typography, colors, and layout in every section.
- **Conversion Focused**: Includes comparison tables, trust bars, and transformation widgets.

## 🛠 Custom Sections & UI Components

The following custom sections have been developed to provide a unique and interactive shopping experience:

### 1. Skincare Transformation Widget (`custom-skincare-transformation.liquid`)
- **Purpose**: Showcases clinical or consumer study results to build credibility.
- **UI Details**: Features animated SVG progress circles that fill as the user scrolls, paired with before/after images and customer testimonials.

### 2. Product Comparison Table (`custom-comparison.liquid`)
- **Purpose**: Highlights brand advantages against competitors.
- **UI Details**: A clean, side-by-side grid comparing features with "check" and "cross" icons to visually emphasize product superiority.

### 3. Product & Video Widget (`custom-product-widget.liquid`)
- **Purpose**: Engages users with product demonstrations alongside a direct purchase path.
- **UI Details**: A split section with an auto-playing video (YouTube/Vimeo/MP4) on one side and a product card or carousel on the other.

### 4. Custom FAQ System (`custom-faq.liquid`)
- **Purpose**: Addresses common customer concerns in an organized manner.
- **UI Details**: Collapsible accordion-style layout with support for category grouping and rich text content.

### 5. "How It Works" Guide (`custom-why-not-work.liquid`)
- **Purpose**: Explains simple routines or product usage steps.
- **UI Details**: A 3-step grid featuring large, semi-transparent background numbers (01, 02, 03) and card-based typography for a clear instructional flow.

### 6. Premium Benefits Trustbar (`custom-trustbar2.liquid`)
- **Purpose**: Establivals value propositions at a glance.
- **UI Details**: A 4-column icon grid with smooth hover lifting effects. On mobile, it transforms into a horizontal scroll with navigation buttons.

### 7. Premium Feature Showcase (`custom-premium-features.liquid`)
- **Purpose**: Highlights core product benefits with high-quality imagery.
- **UI Details**: Combines a large focal image with a "floating" testimonial card and a vertical stack of benefit cards.

### 8. Flexible Hero Banner (`custom-hero.liquid`)
- **Purpose**: First-touch high-impact storytelling on the landing page.
- **UI Details**: Full-screen or container-width banner supporting video backgrounds, animated headlines, and glassmorphism effects.

### 9. Multi-Item Image Banner (`custom-image-banner.liquid`)
- **Purpose**: Displays lifestyle imagery or secondary collections.
- **UI Details**: Flexible grid that supports "small image" blocks floating over a main background, creating a layered, depth-rich visual.

### 10. Press & Media Banner (`custom-press-banner.liquid`)
- **Purpose**: Showcases "As Seen On" or specific media mentions.
- **UI Details**: A structured split banner for high-authority press images and descriptive text.

### 11. Interactive Progress Pointer (`custom-easy-use.liquid`)
- **Purpose**: Visualizes a step-by-step customer journey or process.
- **UI Details**: Features a unique scroll-triggered connection line that dynamically "draws" itself between step circles as the user scrolls down the page.

### 12. Flexible Layout Container (`custom-section.liquid`)
- **Purpose**: A "blank canvas" for advanced store builders.
- **UI Details**: Allows nesting of any theme or app block with full control over flex-direction, alignment, gap, and background media.

## 🎨 Design System

- **Typography**: Uses modern sans-serif and serif pairings defined in theme settings.
- **Color Palette**: Sophisticated use of greens (`#2d6c4f`), neutrals (`#f9f7f2`), and high-contrast blacks for a clean, medical-grade yet luxurious feel.
- **Animations**: Subtle fade-ups, smooth scaling on hover, and interactive SVG animations for progress tracking.
